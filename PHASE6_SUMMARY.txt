USAGE EXAMPLE
================================================================================

Campaign Character Definition:
```ron
(
    characters: [
        (
            id: "hero_knight",
            starts_in_party: true,  // Starts in party
        ),
        (
            id: "recruit_mage",
            starts_in_party: false, // Starts at inn
        ),
    ],
)
```

```bash
cargo run --bin campaign_validator -- campaigns/my_campaign

# Success (≤6 starting):
✓ 3 starting party members (max 6)

# Error (>6 starting):
✗ Too many starting party members: 7 characters have starts_in_party=true, 
  but max party size is 6
```
