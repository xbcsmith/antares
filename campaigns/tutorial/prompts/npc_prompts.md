# NPC prompt assets

These prompts are designed to generate detailed non-player character reference images for the tutorial campaign. Each prompt produces a single, clean, high-quality portrait with realistic detail, strong visual identity, and a clean composition suitable for concept art or later 3D reference work.

## Base style requirements

For every prompt, use:
- DnD-style fantasy aesthetic
- realistic facial detail, clothing, and equipment
- strong character identity and expression
- centered composition
- clean, uncluttered background
- no text, logos, or watermarks
- high resolution and crisp detail
- suitable for character concept art and 3D reference
