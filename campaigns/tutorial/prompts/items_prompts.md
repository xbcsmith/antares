# Item image generation prompts

These prompts are designed for generating clean, isolated reference images for DnD-style items that can later be used as source material for 3D models. Each prompt is written to favor a crisp silhouette, realistic detail, and a transparent background with no shadows or environmental clutter.

## Base style requirements

For every prompt, use:
- DnD-style fantasy aesthetic
- realistic material detail and craftsmanship
- clean, isolated composition
- transparent background
- no shadows
- no background scene
- no text, logos, or watermarks
- centered framing
- high resolution and crisp edges
- suitable for 3D modeling reference
